# 🌐 Blog Web

Este es mi Blog personal desarrollado con **HTML y CSS**, donde presento mis gustos, habilidades, proyectos y formas de contacto.

---

## 📌 Descripción

El objetivo de este proyecto es mostrar mis conocimientos en desarrollo web front-end, aplicando buenas prácticas de diseño, estructura semántica y diseño responsivo.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design

---

## 📂 Secciones del proyecto

- **Inicio** → Presentación personal
- **Sobre mí** → Información sobre mí y mis intereses
- **Habilidades** → Tecnologías y nivel de dominio
- **Proyectos** → Trabajos realizados
- **Contacto** → Formulario e información de contacto

---

## 🎯 Características

- Diseño moderno y limpio
- Layout responsivo (adaptable a móviles)
- Uso de Flexbox y Grid
- Componentes reutilizables
- Buenas prácticas en HTML semántico
